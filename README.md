# Full-Stack Diagrams

Generate SVGs for pieces that can be used to create full-stack diagrams for 
sub-second acceptance tests.

## Edit and build SVGs pieces

The pieces are defined in `pieces.txt`. Rebuild the `./svg/*.svg` files:

    yarn build
